# Commandes slash disponibles

## /new-screen [nom]
Génère un écran Compose complet avec son ViewModel et ses états UI.
Lit : .claude/skills/android-kotlin.md + .claude/skills/ux-interface.md + .claude/skills/apple-design.md

## /new-usecase [nom]
Génère un UseCase dans domain/usecase/ avec son interface repository si nécessaire.
Lit : .claude/skills/android-kotlin.md

## /new-endpoint [méthode] [chemin]
Génère une route FastAPI avec son schéma Pydantic et sa logique de base.
Lit : .claude/skills/backend-python.md

## /heatmap-debug
Affiche les données de la grille IDW sous forme lisible pour déboguer.
Lit : .claude/skills/heatmap-plan.md

## /check-ux
Vérifie qu'un écran Compose respecte les règles UX (vocabulaire, accessibilité, tap targets).
Lit : .claude/skills/ux-interface.md + .claude/skills/apple-design.md

## /apple-design [composant]
Génère ou refactorise un composant Compose en appliquant les tokens Apple (AppColors, AppType, AppShape, AppMotion).
Lit : .claude/skills/apple-design.md
